ICMP Timestamp (-PP):
Sends a request asking "What time is it?" to the target.
If the target responds, it reveals its system time.
Used to check if host is live and sometimes to measure clock differences.

ICMP Address Mask (-PM):
Sends a request asking "What is your subnet mask?" to the target.
If the target responds, it gives its network mask.
Used to check if host is live and sometimes to learn about network setup.
Summary:

ICMP Echo Scan (-PE)

Sends ICMP Echo requests (ping).
Checks if the target replies (classic ping).
Most common for host discovery.

-PP: Checks if host replies with time info.
-PM: Checks if host replies with subnet mask info.
